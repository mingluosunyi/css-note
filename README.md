# css-note
## 第一章：层叠
### 层叠规则
样式表的来源 -> 选择器的优先级 -> 源码顺序

#### 样式表的来源
根据优先级分别是：
+ 作者样式表(前端)
+ 用户样式表(用户)
+ 用户代理样式表(浏览器)

#### 选择器的优先级
+ 行内样式
+ id
+ 类
+ 标签
> 伪类选择器和属性选择器与一个类选择器的优先级相同，通用选择器和组合选择器对优先级没有影响
>
> 伪类选择器的顺序是 "Love/Hate" 原则：link，visit，hover，active

#### 源码顺序
后面的代码会覆盖前面的

### 继承
文本，列表和表格的边框属性会被继承。

### 简写属性
#### 简写属性的危险⚠️
大多数简写属性可以省略一些值，但这些值依旧会被设置为默认属性，因此可能覆盖优先级低的样式。因此要当心！
#### 简写属性的顺序
+ 四个值：上，右，下，左。
+ 两个值：水平，竖直
